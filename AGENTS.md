# Guía para Agentes de IA

Este repositorio contiene las especificaciones y herramientas para la implementación del proyecto Travesía en Moodle. Si eres un agente de IA que está comenzando a trabajar en este proyecto, sigue esta guía de navegación para comprender el contexto y acceder a la documentación relevante usando los enlaces relativos a continuación.

## Documentos Principales de Planificación e Implementación

- **Plan de Implementación General (Obligatorio):**
  Consulte el archivo [Plan de Implementación V3.1 Completo.md](Plan%20de%20Implementacion%20V3.1%20Completo.md) para conocer todo lo que debe implementarse de forma mandatoria.
- **Contenidos Opcionales (No leer por ahora):**
  El archivo [Plan de Implementación V3 - Contenidos opcionales.md](Plan%20de%20Implementacion%20V3%20-%20Contenidos%20opcionales.md) contiene implementaciones opcionales que no deben procesarse ni implementarse por el momento.
- **Restricciones y Lecciones Aprendidas (Crucial):**
  Antes de realizar cualquier cambio o desarrollo, lea atentamente [constraints.md](constraints.md) para conocer las restricciones de base de datos, reglas de negocio, mapeos en Moodle y soluciones de estilo CSS.
- **Configuración del Sistema:**
  La configuración de URLs, cmids, dataids, preguntas y umbrales se encuentra detallada en [config_2.md](config_2.md).

## Datos y Estructura de Referencia

- **Recolección de Datos:**
  En [Recoleccion de datos 3.md](Recoleccion%20de%20datos%203.md) se encuentran todos los datos recopilados necesarios para la creación de los scripts.
- **Estructuras de Bases de Datos:**
  Si necesitas comprender mejor el esquema de datos o resolver dudas específicas sobre la base de datos, revisa [Estructuras de DB's.md](Estructuras%20de%20DB's.md) (ten en cuenta que la fuente de verdad principal sobre los campos siguen siendo los HTML de Moodle).
